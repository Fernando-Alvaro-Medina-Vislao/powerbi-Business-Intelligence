## 📊 Business Intelligence – Caso Alicorp (Power BI)

Dashboard interactivo de Business Intelligence desarrollado en Power BI, orientado al análisis de indicadores empresariales mediante KPIs y visualizaciones dinámicas para apoyar la toma de decisiones estratégicas.

## 📊 Objetivo del proyecto

Analizar la información empresarial de Alicorp para identificar:

* Eficiencia operativa en producción
* Cumplimiento de entregas (logística)
* Rentabilidad por producto
* Desempeño de ventas
* Satisfacción del personal
* Indicadores de sostenibilidad

---

## 🛠️ Herramientas utilizadas

* Power BI Desktop
* DAX
* SQL Server
* Modelado de datos (Star Schema)
* ETL / Integración de datos

---

## 📌 Principales métricas

* OEE (Eficiencia operativa)
* OTIF (Entregas a tiempo)
* Margen de rentabilidad
* Ingreso por canal
* Índice de satisfacción del empleado
* Indicadores ambientales
